Feature Manipulate Engineer (FME)

This certificate verifies the completion of a four-hour introductory course designed to familiarize learners with the core components of the FME Platform. The curriculum covers foundational knowledge of what FME is and how it operates, followed by hands-on experience in exporting, transforming, and integrating data across multiple systems and formats

During the course, participants learn to:

  • Connect diverse data sources and applications using FME’s no-code interface
  • Design and execute basic workflows for data translation, transformation, and automation
  • Apply FME tools such as Readers, Writers, and essential Transformers
  • Automate repetitive data tasks using visual workflows and understand FME's scheduling options

This training provides a practical foundation in FME workflows and offers essential skills for automating ETL operations in spatial and non-spatial data contexts.
