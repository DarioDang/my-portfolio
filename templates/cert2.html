Machine Learning for Data Analytics

This certificate demonstrates proficiency in applying machine learning techniques to data analysis using BigQuery ML. The program covered the complete ML pipeline—from data preparation and attribute selection to model building and evaluation. Key topics included linear and logistic regression, decision tree ensembles (Random Forest, Boosted Tree), clustering (KMeans), time series forecasting (ARIMA), and deep learning techniques. Emphasis was placed on repeatable dataset splitting, feature engineering, and hyperparameter tuning to ensure robust and scalable models.
